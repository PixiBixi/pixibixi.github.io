    Installer une version custom de PHP

    Warning

    Aujourd'hui, nous sommes en PHP 8.x, la méthode reste toujours valable, mais renseignez-vous sur la version de PHP

    Ajout du repo :

    apt-get install apt-transport-https lsb-release ca-certificates
wget -O /etc/apt/trusted.gpg.d/php.gpg https://packages.sury.org/php/apt.gpg
echo "deb https://packages.sury.org/php/ $(lsb_release -sc) main" > /etc/apt/sources.list.d/php.list

    Puis un petit update/install php8.x, et le tour est joué

    apt update && apt install -y php8.2-fpm
