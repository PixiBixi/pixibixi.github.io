# Effectuer un Speedtest depuis son serveur 
 
## Introduction 
 
Pour effectuer simplement un speedtest sur votre serveur, installer 
speedtest-cli 
 
``` bash 
$ apt install speedtest-cli 
``` 
 
Puis lancer simplement la commande 
 
``` bash 
λ jeremy ~ → speedtest-cli
``` 
